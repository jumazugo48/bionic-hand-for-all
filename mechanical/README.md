# Mechanical

This folder contains all 3D design files for the
Bionic Hand For All project.

## Contents

- CAD source files (Fusion 360)
- STEP files (universal CAD exchange format)
- STL files ready for 3D printing

## Manufacturing Methods

| Component | Method |
|---|---|
| Structural parts | FDM 3D printing in PETG |
| Finger joint pins | Lathe-machined steel |
| Finger joint rings | Lathe-machined steel |
| Joint retainers | Commercial E-clip hardware |
| Arm attachment | Elastic fabric straps |

## Print Settings (recommended)

To be defined during prototyping phase.

## Software Required

| File type | Software |
|---|---|
| `.f3d` | Fusion 360 (free for students) |
| `.step` | Fusion 360, FreeCAD (free) |
| `.stl` | Any slicer: Cura, PrusaSlicer |
