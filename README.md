## Project Overview

Design and fabrication of a functional low-cost bionic hand prosthesis
controlled by electromyographic (EMG) signals. The finished prosthesis
will be donated to a person in need. All designs, code and documentation
are released as open source to enable global replication.

## Core Principles

- **Donate, never sell.** The physical prosthesis produced with this
  design must always be given free of charge to the recipient.
- **Open by design.** Every file needed to replicate this project
  is published here.
- **Community-driven.** Once published, the community takes over.
  No ongoing support is expected from the original author.

## Project Structure

This repository has two pillars:

### 1. Technical Development — The Prosthesis
Design and fabrication of a transradial upper limb prosthesis.
Functional, bionic, low-cost and fully replicable.

### 2. Institutional Initiative — The Laboratory
Formal proposal submitted to school authorities for the creation
of a permanent Bionic Prosthetics Development Lab at EIDFS.

## Objectives

- Design and build a functional EMG-controlled prosthetic hand
- Donate the finished prosthesis to a person in need
- Establish an institutional lab for continued future development
- Provide open source documentation for global replication
