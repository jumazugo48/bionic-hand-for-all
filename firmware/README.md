# Firmware

This folder contains all code for the ESP32 microcontroller
of the Bionic Hand For All project.

## Development Environment

- **IDE:** VS Code + PlatformIO extension
- **Microcontroller:** ESP32 (ESP-WROOM-32)
- **Framework:** Arduino (via PlatformIO)

## Features

- Real-time EMG signal acquisition and processing
- 6 predefined grip modes
- User-customizable grip storage
- (Planned) Bluetooth telemetry
